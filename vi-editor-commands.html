Vi Editor commands


ben - Fr 18 September 2009 - Command reference

**Quit & Save**
:w Save
:x Save & exit
:q Quit if there have been no changes
:q! Quit & discard changes
**Insert / Edit**
i Insert mode
I Insert at beginning of line
A Append to end of line
R Replace mode
r Replace a single character
Esc Exit insert or replace mode
u Undo last change
U Undo all changes to line
**Navigate**
k Line up
j Line down
10j Ten lines down
h Move left
l Move right
w Go to next word
b Go to previous word
0 Go to beginning of line
\$ Go to end of line
G Go to end of file
g Go to beginning of file
nG Go to nth line of file
ctrl-f jump forward one screen
ctrl-b jump backwards one screen
**Delete, Cut & Paste**
dw Delete/cut word
x Delete/cut character at cursor position
X Delete/cut character left to cursor
D Delete/cut to end of line
dd Delete/cut line
10dd Delete/cut ten lines
v Select at cursor
V Select line
yw Copy word
yy Copy line
y Copy selection
P Paste
**Search & Replace**
/ Search forwards
? Search backwards
n Jump to next search result
N Jump to previous search result
:%s/pattern/string/flag Replace pattern with string according to flag:
g Flag - Replace all occurences of pattern
c Flag - Confirm single replaces
