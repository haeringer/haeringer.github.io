Packet capturing on Cisco ASA

Do 17 Februar 2011
by ben
in Networking

tagged ASA, Cisco IOS, Packet capturing

The ASA5500 firewalls offer a great and simple packet capturing tool to see if specific packets pass a certain interface.

First write an access list that defines the source and destination address of the packets you want to see:

access-list CAPACL-TO-CME extended permit udp any host 10.1.1.1

Set the capture, referencing to the access list and defining the interface on which you want to capture:

capture CAP-TO-CME access-list CAPACL-TO-CME interface inside

Initiate some of the traffic of interest and see if it got captured:

show capture CAP-TO-CME

238 packets captured
1: 18:04:05.800938 192.168.3.10.5060 > 10.1.1.1.5060:  udp 494 
2: 18:04:06.971247 192.168.2.10.5060 > 10.1.1.1.5060:  udp 446
