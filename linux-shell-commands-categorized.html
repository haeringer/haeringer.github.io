Linux shell commands (categorized)

Mi 09 September 2009
by ben
in Linux

tagged Command reference, Shell

**Help**
*command*--help show a brief help text
man *command* show the man page of a command
**Files and directories**
cat display the contents of a file
cd change to the home directory
cd *directory* change to this directory
cd .. change one directory up
cd / change to the main directory
cd - change to the last directory
cmp compare two files
cp copy file(s)
file determine file type
gzip / gunzip compress or decompress file(s)
tar xvf extract \*.tar.gz file
tar xvjf extract \*.tar.bz2 file
ls show contents of directory
ls -d \*/ show only directories within a directory
mv move or rename files or directories
mkdir create a directory (mkdir -p for a hierarchy)
pwd display the working directory (Print Working Directory)
rcp copy files between machines (scp for secure copy via ssh)
rm remove files
rm -r remove directory + contents
less display output and page through text one screenful at a time
ln -s create a symbolic link to a file (ln -h for a hardlink)/directory
split split a file, e.g. for transfer
sum show file size and checksum
touch create a file
**Rights management**
chgrp change group ownership
chmod change access permissions, change mode
chown change file owner and group
**Users & groups**
chfn change the user information of an account
chsh change the login shell of a user account
finger show user account information like name, login time etc.
groupadd add a new user group
groupdel delete a user group
groupmod modify a user group
groups show group name a user is in
id show real and effective user id (uid) and group id (gid)
last show the last logins
newgrp change the current group
passwd set or change the password
su open a new session with another user account
sudo execute a command with root rights, if granted
useradd add a new user (-m to additionaly create the home dir)
userdel delete a user
usermod modify a user account
w show the currently logged in users (similar: who)
**Search**
find / -name xyz search a folder hierarchy (root in this case) for file "xyz"
locate search the system for files
updatedb update the database before using locate
grep search within a file (cat test.txt | grep whatever OR grep whatever test.txt)
which show path of binary of a program
**Network**
ftp open an ftp session
ifconfig show network interfaces including current IP address
ifup / ifdown start / stop a network interface
ipchains tool for configuration of the kernel firewall
netstat show information about network connections, routing tables etc.
ping send ICMP packets to an IP address
route create static routes
telnet open a telnet session
traceroute trace the route to an IP address
ssh open an SSH session
**Partitions / File system**
badblocks search a partition for bad blocks
dd copy partitions / file systems
dumpe2fs show information about the file system on a partition
e2fsck check and repair ext2 file systems
e2label show or assign a label to a file system
fdisk -l list all partitions, including e.g. connected USB drives
fsck check and repair linux file systems
mkdosfs create a Dos (FAT16 & FAT32) file system
mke2fs create an ext2 file system
mkfs frontend for several file system creators
mkswap create a swap partition
mount / umount mount / unmount a file system
**System**
apt-cache search search the repository for software packages (Debian)
apt-get install/remove install or remove software packages (Debian)
df -h disk free - display free disk space
du -h disk usage - show size of a file or directory
free display RAM usage
kill *PID* stop a process
killall stop processes by name
nice change the priority of a process
nohup run a command immune to hangups
pidof show the PID of a process
ps ax show all processes with their PID, user ID, terminal ID etc.
pstree show all processes as a tree
top show processes in real time (press c for process paths and options
uptime show uptime of the machine
crontab show the user specific cron table / open it for editing
halt shutdown the system (same as shutdown -h now)
history show the last commands issued by the user
reboot reboot system (same as shutdown -r now)
uname -mrs show linux OS version
