Transfer PuTTY settings

Fr 04 Dezember 2009
by ben
in Windows

tagged PuTTY

If you would like to transfer your saved PuTTY configuration to another Windows machine, you might wonder where this configuration is saved. PuTTY does not use an INI file or something similar, but stores its settings in the Registry.

Open the registry by running regedit and search for "SimonTatham" (the name of the original developer). You will land at HKEY_CURRENT_USERSoftwareSimonTatham.

Now choose File -> Export and save the settings as e.g. puttysettings.reg

The next time you want to apply the settings, just doubleclick this file.
